# Farming system
This file outlines the set of cards used for the farming game loop.

#### Subsistence Farming (Technology)
- Cost: 3W, 2G
- Effect:
	- Place 1 Farm on board
	- Shuffle 2 Farm cards into deck
	- Shuffle Granary into deck
#### Granary (Building)
Cost: 3W
Effect:
- Adjacent Farms gain:
	- +1 Food on Harvest
	- 20% chance to shuffle a Farm card into deck on Harvest
- Shuffle Three-Field Rotation into deck

#### Three-Field Rotation (Technology)
Cost: 4W, 4G
Effect:
- All Farms produce +1 passive Food per turn
- Harvest on farm produces +2 Food
- Shuffle 2 Farm cards into deck
- Shuffle Advanced Irrigation into deck

#### Advanced Irrigation (Technology)
Cost: 6W, 6G
Effect:
- Build 1 Aqueduct on target tile
#### Aqueduct (Building)
2 Villager slots
Effect:
- When harvesting an adjacent Farm AND Villager slot available
   - Draw 1 card
